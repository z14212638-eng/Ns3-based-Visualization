# Ns-3 PPDU Timeline Visualizer

A Qt-based visualization tool for **ns-3 PHY/MAC layer PPDU activity**, designed to provide an **interactive, time-aligned timeline view** of wireless transmissions.  

This project focuses on **PPDU-level visualization and analysis**, including:
- Multi-AP timelines
- Overlapping transmissions
- Hover-based inspection
- Time-range selection with automatic zoom
- Throughput / channel utilization statistics

It is intended as a **research and debugging tool** rather than a full simulator or NetAnim replacement.

- MPDU aggregation count

---

## ðŸ“ˆ Metrics Definition

For a selected time range `[T_start, T_end]`:

- **Busy Time**  
  Sum of PPDU durations overlapping the interval

- **Utilization**  
  ```
  busy_time / (T_end - T_start)
  ```

- **Throughput (Mbps)**  
  ```
  (total_bits_transmitted) / duration
  ```

Overlaps are handled proportionally by time intersection.
