# Simple Simulation first.cc
first.cc is a simple simulation example in ns3 tutorial.
## Simulation Overview
```
Default Network Topology

        10.1.1.0
n0 -------------------- n1
      point-to-point

```

## Simulation Configuration 
| Parameter | Value |
| --- | --- |
| SniffPPDU | False |
| Number of nodes | 2 |
| Node IP | 10.1.1.0 |
| Link bandwidth | -- |
| Link delay | 2ms |
| Packet size | 1024 bytes |
